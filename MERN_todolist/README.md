# TodoList MERN

Application MERN pour gérer vos tâches.

## Installation rapide

### Backend

```bash
cd backend
npm install
```

Créer `backend/.env` :

```env
MONGODB_URI=mongodb://localhost:27017/mern-todo
PORT=5000
```

### Frontend

```bash
cd frontend
npm install
```

## Démarrage

**Terminal 1 - Backend :**

```bash
cd backend
npm run dev
```

**Terminal 2 - Frontend :**

```bash
cd frontend
npm run dev
```

Ouvrir **http://localhost:5173**

## Stack

- MongoDB + Express + React + Node.js
- Architecture MVC
- REST API

## Fonctionnalités

- Ajouter des tâches
- Modifier des tâches
- Marquer comme complété
- Supprimer des tâches
- Design responsive
